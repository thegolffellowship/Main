# The Golf Fellowship

Membership portal for The Golf Fellowship.

## Features

- **Membership Sign-In Page** - Clean, responsive sign-in form with:
  - Email and password validation
  - Password visibility toggle
  - Remember me functionality
  - Google sign-in option
  - Mobile-responsive design

## Project Structure

```
Main/
├── index.html      # Sign-in page
├── css/
│   └── styles.css  # Styling
├── js/
│   └── signin.js   # Form validation
└── README.md
```

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser

No build tools or dependencies required.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
