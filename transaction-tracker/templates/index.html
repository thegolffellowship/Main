Connector / Webhook ▾

External systems can push order data to this tracker via the webhook endpoint.

Endpoint: —
Method: POST
Auth: X-API-Key: <your CONNECTOR_API_KEY>
Example — push structured items: 
{
  "items": [{
    "email_uid": "ext-001", "item_index": 0,
    "merchant": "The Golf Fellowship",
    "customer": "John Doe",
    "order_id": "R123456",
    "order_date": "2026-02-11",
    "item_name": "Feb 22 - LaCANTERA",
    "item_price": "$158.00",
    "chapter": "San Antonio",
    "side_games": "NET Points Race"
  }]
}
Example — send raw email text for AI parsing: 
{
  "raw_email": {
    "uid": "ext-002",
    "subject": "New Order #R999",
    "from": "noreply@mysimplestore.com",
    "text": "... full email body ..."
  }
}
